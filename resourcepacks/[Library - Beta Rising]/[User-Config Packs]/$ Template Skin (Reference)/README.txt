You might have noticed your skin is gone! This is because the mod "Random Tweaks" intercepts Mojang's online skin system and overrides the player skin to force Steve (or optionally Alex), for various reasons. By extension, it enables easier and completely offline skin support for Minecraft.
--------------------------------------------------------------------------------------------------------------------------------
Note: Instead of using a resourcepack to change your skin, press 'G' in-game to access the mod 'Custom Player Models' and select 'Models' to browse a fully stocked library overflowing with built-in skins, ranging from legacy console skins to beloved IPs, modern defaults, and more!

--------------------------------------------------------------------------------------------------------------------------------
To use a custom skin:

1) Find any skin texture

2) Place it at this location with the corresponding name to match:

"[resource_pack_name]/assets/minecraft/textures/entity/player/wide/steve.png" 

OR...

"[resource_pack_name]/assets/minecraft/textures/entity/player/slim/alex.png" 

3) Enable the resourcepack
